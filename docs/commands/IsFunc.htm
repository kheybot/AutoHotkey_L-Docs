IsFunc(FunctionName) [v1.0.48+]

Check if FunctionName exist explicitly in the script.

If FunctionName does not exist explicitly in the script (by means such as #Include or a non-dynamic call to a library function), IsFunc() returns 0. Otherwise, it returns one plus the minimum number parameters (e.g. 1 for a function that requires zero parameters, 2 for a function that requires 1 parameter, etc.).

For example, the statements if IsFunc("MyFunc") and if IsFunc(VarContainingFunctionName) would be true if the function exists, and false otherwise. In v1.1.00+, FunctionName can be a function reference instead of a name.

Functions, Dynamic function-call, A_ThisFunc

Example

MsgBox % IsFunc("Test") ; will return 2

Test(var) { ; sample function
    Return var++
}
